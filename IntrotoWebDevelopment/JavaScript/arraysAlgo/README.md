Using the given array:\
var testArr = [6,3,5,1,2,4]\

Print Values and Sum\

The expected output will be:\ 
Num 6, Sum 6\
Num 3, Sum 9\
Num 5, Sum 14\
Num 1, Sum 15\
Num 2, Sum 17\
Num 4, Sum 21\

[0,3,10,3,8,20]\
